Welcome in the Folder Monitor Application (FMA)

FMA is a microservice which analyzes the files contained in the folder "src/main/resources/checkFolder"

There are two REST Web API:

  1. /fmonitoringsys/api/filesinfo?start=YOUR_START_DATE_TIME&end=YOUR_END_DATE_TIME :
    give in output all the info of the files (one, more o noone) contained in the audit folder and that have as last change date in the given datetime range

  2. /fmonitoringsys/api/file?md5=YOUR_MD5_CHECKSUM :
    give in output the file (only one, or none) information contained in the audit folder, and which has as Hash MD5 codification equals to the given input parameter

Warning to the dateTime: the format shoud be yyyy-mm-ddThh:mm:ss

Here below some examples which have consistent and not void results:

  1. http://localhost:8080/fmonitoringsys/api/filesinfo?start=2015-09-26T01:30:00.000&end=2030-09-26T01:30:00.000
  2. http://localhost:8080/fmonitoringsys/api/file?md5=d41d8cd98f00b204e9800998ecf8427e"

Here below some examples which have no result in the monitoring request:

  1. http://localhost:8080/fmonitoringsys/api/filesinfo?start=2015-09-26T01:30:00.000&end=2016-09-26T01:30:00.000
  2. http://localhost:8080/fmonitoringsys/api/file?md5=xxxxx
